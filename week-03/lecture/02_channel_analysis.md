# 流入チャネルの読み方

## 1. チャネルとは何か

「チャネル（Channel）」とは、ユーザーがサイトに訪問した**経路（ルート）**のことです。
どこから来たユーザーが最も購入するか、最もコンバージョン率が高いかを把握することで、
マーケティング予算の最適化につながります。

---

## 2. GA4 の主要チャネル一覧

### Organic Search（自然検索）
- **意味**: Google/Yahoo などの検索エンジンで検索し、広告以外の検索結果からアクセス
- **特徴**: 広告費ゼロ・長期的な集客・SEO 施策の成果が反映される
- **例**: 「ハンドメイドアクセサリー 通販」で検索してクリック

### Paid Search（有料検索）
- **意味**: Google 広告・Yahoo! 広告などのリスティング広告からのアクセス
- **特徴**: 費用がかかるが即効性がある・UTM パラメータが自動付与される
- **例**: Google 広告の「ハンドメイド アクセサリー」キーワード広告をクリック

### Direct（直接アクセス）
- **意味**: URL を直接入力・ブックマーク・参照元不明のアクセス
- **特徴**: 実は「参照元不明」が多く含まれる・ブランド認知度の高さが反映される
- **例**: ブックマーク登録済みサイトへのアクセス

### Referral（参照サイト）
- **意味**: 他のウェブサイトのリンクからのアクセス（SNSを除く）
- **特徴**: ブログ・ニュースサイト・他のECサイトからのリンク
- **例**: ファッションブログから商品ページへのリンクをクリック

### Organic Social（自然ソーシャル）
- **意味**: SNS の投稿リンク（広告ではない）からのアクセス
- **特徴**: Instagram・Twitter・Facebook などの通常投稿
- **例**: インスタグラムの商品紹介投稿からサイトへ遷移

### Paid Social（有料ソーシャル）
- **意味**: SNS 広告からのアクセス
- **特徴**: Facebook 広告・Instagram 広告・Twitter 広告など
- **例**: Instagram のショッピング広告をクリック

### Email（メール）
- **意味**: メールマガジンなどのメールリンクからのアクセス
- **特徴**: 既存顧客へのリテンションマーケティングが反映される
- **例**: ニュースレターの「新着商品を見る」リンクをクリック

## 3. UTM パラメータの仕組み

### UTM パラメータとは

URL の末尾に付けることで、GA4 に「どのキャンペーンから来たか」を
教えるための識別タグです。Google Analytics が開発したため頭文字が UTM
（Urchin Tracking Module）になっています。

### UTM パラメータの種類

| パラメータ | 意味 | 例 |
|---|---|---|
| `utm_source` | 流入元（どのサービスか） | `google`, `instagram`, `newsletter` |
| `utm_medium` | 媒体種別（どんな手段か） | `cpc`, `social`, `email` |
| `utm_campaign` | キャンペーン名 | `summer-sale-2026`, `new-arrival` |
| `utm_content` | 広告バリエーション（A/Bテスト用） | `banner-a`, `text-link` |
| `utm_term` | 検索キーワード（Paid Search 用） | `handmade-accessories` |

### UTM パラメータ付き URL の例

```
https://example.com/products/necklace?
  utm_source=instagram
  &utm_medium=social
  &utm_campaign=summer-sale-2026
  &utm_content=story-banner
```

このURLをクリックしたユーザーは GA4 で以下のように分類されます。

| GA4 項目 | 値 |
|---|---|
| セッションのメディア | social |
| セッションのソース | instagram |
| セッションのキャンペーン | summer-sale-2026 |

### UTM パラメータがないとどうなるか

UTM パラメータが付いていないリンクは、SNS 投稿であっても
GA4 が「Direct」と判定してしまう場合があります。
正確なチャネル計測のためには、外部からのリンクには必ず UTM を付けます。

## 4. チャネル分析で見るべき指標

### 量的指標（どれだけ集客できているか）
- **セッション数**: チャネルごとの訪問数
- **ユーザー数**: チャネルごとのユニークユーザー数

### 質的指標（集客の質はどうか）
- **エンゲージメント率**: 「エンゲージメントのあったセッション / 全セッション」の割合
  - 高い = サイトに関心を持ったユーザーが多い
- **コンバージョン率**: 購入・登録などの目標達成率
- **セッションあたりのコンバージョン数**

### 注目すべきパターン

| パターン | 解釈 |
|---|---|
| Organic Search が多い | SEO が効いている |
| Direct が異常に多い | UTM 未設定のキャンペーンがある可能性 |
| Paid Search のコンバージョン率が低い | ランディングページとキーワードのミスマッチ |
| Referral が急増している | どこかのサイトに紹介された |

---

## 5. チャネルとファネルを組み合わせる

チャネル分析とファネル分析を組み合わせると、より深い洞察が得られます。

**質問例**:
-「Organic Search から来たユーザーと Paid Social から来たユーザーでは、どちらがカート追加率が高いか？」
- 「メール経由のユーザーは購入完了まで至る割合が高いか？」

このような分析を行うには、GA4 の探索レポートでセグメントを使います（handson 03 で実施）。

## まとめ

| チャネル | 特徴 | 施策の方向性 |
|---|---|---|
| Organic Search | 長期的・費用ゼロ | SEO コンテンツ・キーワード対策 |
| Paid Search | 即効性・費用あり | 入札最適化・LP 改善 |
| Direct | ブランド認知 | リピーター施策・メルマガ |
| Referral | 他サイトからの信頼 | パートナーシップ・PR |
| Organic Social | SNS での拡散 | コンテンツマーケティング |
| Email | 既存顧客へのリーチ | セグメント配信・パーソナライズ |

次の handson では、実際に GA4 のトラフィック獲得レポートを開いて
各チャネルのデータを読み取っていきます。
