**このハンズオンは、以下の条件を満たしたときに完了とします。**

> そのURLをブラウザで開くと KPI ダッシュボードが表示されること。

## 前提条件（始める前に確認）

- [ ] Week 1〜3 の handson が全て完了している

## よくある質問

**Q: GA4 にデータがなくてグラフが空です**

A: 次のいずれかで対処してください。
1. `curriculum/week-01/handson/` の手順を再確認し、計測が正しく動作しているか確認する
2. GA4 デモアカウントをデータソースとして使って練習する（データが豊富にある）

**Q: ファネルのイベント（view_item 等）が指標一覧に表示されません**

A: GA4 にカスタムイベントとして記録されていないと表示されません。
Week 2 の `01_event_design.md` と `02_implementation.md` を確認してください。

**Q: ダッシュボードの URL を渡したのにクライアントが「アクセスできない」と言われました**

A: 共有設定が「閲覧者」になっているか確認してください。
`docs/portfolio_guide.md` の「URL 共有設定」セクションを参照してください。
